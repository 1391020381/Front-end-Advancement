* html语义化就是根据 结构化的内容,选择合适的标签。

# 为什么要做语义化
* 合适的标签 是内容表达的高度概括,这样浏览器爬虫或者任何机器在读取HTML时,都能更好地理解,进而解析效率更高。
  - 有利于SEO
  - 开发维护体验更好
  - 用户体验更好(如使用alt标签用于解析图片信息)
  - 更好的accessibility,方便任何设备解析(如盲人阅读器)
