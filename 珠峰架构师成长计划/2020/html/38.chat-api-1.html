  • 用户
    • post /login 登录
    • post /validate 验证用户身份
  • 房间
    • post /addRoom 添加房间
    • get /getAllRooms 查看房间列表
  • 消息
    • addMessage 添加消息
    • getAllMessages 获得所有消息
